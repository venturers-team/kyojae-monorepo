# 폐기: 구 Phase 7 — 외부 노트 동기화

## 폐기된 내용

원래 마이그레이션-계획.md v1.0 (2026-04-29 작성)의 구 Phase 7에는 사용자 개인 외부 노트 시스템과 Notion에 대한 동기화 task 5개가 있었음. 구체적 경로·파일명 인용은 외부 컨텍스트 노출 최소화 위해 본 archive에서 생략. 원본은 git history에서 조회 가능 (커밋 이전 버전).

## 폐기 사유

본 monorepo는 GitHub public repo로 운영되는 **자립형 독립 프로젝트**다. 외부 노트 시스템·개인 카테고리·다른 프로젝트와의 cross-link을 본 repo의 task로 등록하는 것은 다음 문제를 야기.

- **독립성 훼손**: 외부 시스템 구조에 의존하면 외부가 바뀔 때마다 본 repo도 갱신해야 함
- **개인 컨텍스트 노출**: public repo에 개인 노트 카테고리·구조가 텍스트로 남음 (사용자명·도구·습관 등)
- **책임 경계 흐림**: 외부 노트 갱신은 사용자 개인 워크플로우, monorepo 마이그레이션의 성공 여부와 무관

→ 외부 동기화는 사용자가 별도로 처리. 본 repo의 phase·task에서는 제거.
